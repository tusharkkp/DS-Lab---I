# DS-Lab---I

A comprehensive collection of **Data Structures** implementations in C++. This repository contains well-structured code examples demonstrating fundamental data structure concepts and algorithms.

## 📋 Contents

This repository includes implementations of various data structures such as:

- **Arrays & Vectors**
- **Linked Lists** (Singly, Doubly, Circular)
- **Stacks**
- **Queues** (Simple, Circular, Priority)
- **Trees** (Binary Search Trees, AVL Trees, etc.)
- **Graphs**
- **Hash Tables**
- **Heaps**
- **Sorting Algorithms**
- **Searching Algorithms**

## 🛠️ Technologies Used

- **Language**: C++
- **Standard**: C++11 or later

## 🚀 Getting Started

### Prerequisites

- A C++ compiler (g++, clang, or MSVC)
- Git

### Compilation

To compile any of the C++ files:

```bash
g++ -o output_file source_file.cpp
./output_file
```
